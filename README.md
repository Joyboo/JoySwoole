
#### 基于 [EasySwoole](https://github.com/easy-swoole/easyswoole) 

#### 文档参考 [Doc](https://github.com/easy-swoole/doc)

- php >= 7.2
- swoole >= 4.4.13


**克隆项目**
> git clone https://github.com/Joyboo/JoybooSwoole.git

**安装Composer依赖**
> composer install
 
**启动**
> php easyswoole server start
